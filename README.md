# ollama-canvas

A multi-tab, "Canvas-style" coding assistant and AI text editing desktop application built using Tauri, React, and Ollama. This app lets you interface with locally hosted Large Language Models (LLMs) to brainstorm, write code, run agentic workflows, and modify text right within a rich text editor. Everything runs locally on your machine, ensuring complete privacy.

## Features

- **Native Desktop App:** Wrapped in Tauri for a fast, minimal footprint.
- **Local AI Generation:** Connects directly to `localhost:11434` (Ollama) to run models.
- **Rich Text Engine:** Powered by Tiptap. Supports Markdown rendering (Headers, code blocks, lists) dynamically generated by the LLM.
- **Multi-Tab Workspace:** Open and edit multiple files concurrently without losing context.
- **Conversational Interface:** Use the bottom chat bar to write new content or the floating Bubble Menu ("Ask AI to edit") to rewrite specific highlighted text.
- **Dynamic Model Selection:** Automatically detects and lets you switch between installed Ollama models right from the UI.

## Prerequisites

1.  **Ollama**: You must have [Ollama](https://ollama.com/) installed and running locally. You will need to pull at least one model (e.g., `qwen3.5:2b` or `deepseek-coder-v2`).
    ```bash
    ollama run exaone3.5:7.8b
    ```
2.  **Node.js**: Ensure Node.js (v18+) is installed.
3.  **Rust**: Required by Tauri to build the native app shell. 
    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```
4.  **System Dependencies (Linux Only)**: If you are running on a clean Linux environment (like Ubuntu/Debian), you need the WebKit and GTK dev libraries for Tauri:
    ```bash
    sudo apt-get update
    sudo apt-get install -y libwebkit2gtk-4.1-dev build-essential curl wget file libssl-dev libgtk-3-dev libayatana-appindicator3-dev librsvg2-dev
    ```
    *(Note: macOS and Windows users generally only need Node, Rust, and the standard build tools. See the [Tauri prerequisites guide](https://tauri.app/v1/guides/getting-started/prerequisites) for specifics).*

## How to Run (Development)

1.  Clone the repository and install the Node dependencies:
    ```bash
    git clone https://github.com/Igriscodes/ollama-canvas.git
    cd ollama-canvas
    npm install
    ```
2.  Start the application in development mode (this will compile the Rust backend and open the app window):
    ```bash
    npm run tauri dev
    ```

## How to Build for Production

To create an optimized production executable for your current operating system:

```bash
npm run tauri build
```
The compiled binary will be located inside the `src-tauri/target/release/bundle/` directory.
